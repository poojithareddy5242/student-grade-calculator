# Student Grade Calculator

A Python console application that calculates a student's total marks, percentage, and grade across multiple subjects.

## Features
- Takes subject-wise marks as input
- Flags subjects where marks are below 40 (fail warning)
- Calculates overall percentage and assigns a grade (Distinction/A+/A/B/C/D/Fail)

## How to run
Run `main.py` using Python 3. Enter the student's name, number of subjects, and marks when prompted.
